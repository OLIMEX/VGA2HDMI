# VGA2HDMI
VGA to HDMI converter

![VGA2HDMI-1](DOCUMENTS/VGA2HDMI-1.jpg)

## Features

* VGA input connector
* HDMI output connector
* USB-C for power supply
* Power supply status LED
* VGA signal status LED
* HDMI signal status LED
* Dimensions 51x33mm
* 2 mount holes
